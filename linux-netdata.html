Konfigurasi Instalasi Netdata Pada Linux Debian

Konfigurasi ini membahas implementasi dan instalasi sistem pemantauan server menggunakan netdata pada debian 12. netdata berfungsi untuk memantau performa sistem secara real-time, mencakup penggunaan cpu, memori, disk, jaringan, serta proses yang berjalan. melalui konfigurasi ini, server debian dapat menampilkan data operasional secara interaktif melalui antarmuka web yang mudah diakses melalui jaringan lokal. dengan penerapan ini, administrator dapat melakukan pemantauan kondisi sistem secara efisien, mendeteksi masalah kinerja lebih cepat, serta mengoptimalkan penggunaan sumber daya server dengan cara yang fleksibel dan terukur.

Langkah Konfigurasi

1

Persiapan Awal

Sebelum memulai konfigurasi, pastikan perangkat berikut sudah dipersiapkan :

  • Server dengan sistem operasi linux debian
  • Akses root atau user dengan privilege sudo
Gambar 1.1: Linux Debian
Gambar 1.2: Netdata
2

Update & Upgrade Sistem

Langkah pertama adalah update & upgrade sistem agar repository menjadi paling update :

Gambar 2.1: Update Sistem

Commands :

apt update && apt upgrade -y
3

Instal Netdata

Langkah selanjutnya adalah Instal netdata :

Gambar 3.1: Instal Netdata

Commands :

apt install netdata -y
4

Enablekan Netdata

Langkah selanjutnya adalah enablekan netdata agar selalu aktif :

Gambar 4.1: Enablekan Netdata

Commands :

systemctl enable --now netdata
5

Periksa Status Netdata

Langkah selanjutnya adalah memeriksa status netdata :

Gambar 5.1: Periksa Status Netdata

Commands :

systemctl status netdata
6

Masuk Directory Netdata

Langkah selanjutnya adalah masuk directory netdata :

Gambar 6.1: Masuk Directory Netdata

Commands :

nano /etc/netdata/netdata.conf
7

Konfigurasi Directory Netdata

Langkah selanjutnya adalah mengubah bind socket to IP = 127.0.0.1 menjadi bind socket to IP = 0.0.0.0 atau bisa juga menggunakan ip address server :

Gambar 7.1: Konfigurasi Directory Netdata

Commands :

bind socket to IP = 0.0.0.0
8

Restart Netdata

Langkah selanjutnya adalah merestart layanan netdata :

Gambar 8.1: Restart Netdata

Commands :

systemctl restart netdata
9

Instal UFW

Langkah selanjutnya adalah mengistall ufw yang digunakan untuk mengatur firewall :

Gambar 9.1: Instal UFW

Commands :

apt install ufw
10

Konfigurasi Firewall

Langkah selanjutnya adalah konfigurasi firewall untuk mengatur izin port dari netdata :

Gambar 10.1: Konfigurasi Firewall

Commands :

ufw allow 19999/tcp
ufw reload
ufw status
11

Pengujian

Langkah terakhir adalah menguji dengan akses lewat browser http://ip-server:19999/ :

Gambar 11.1: Pengujian

Tips Penting

Pastikan sistem sudah ter-update dengan menjalankan sudo apt update && sudo apt upgrade sebelum memulai konfigurasi.
