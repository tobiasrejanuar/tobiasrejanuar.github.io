Dasar Python

Projek ini bertujuan untuk mempelajari dasar-dasar pemrograman Python melalui praktik sederhana namun aplikatif. Materi yang dipelajari meliputi penggunaan variabel, tipe data, operator, percabangan, perulangan, fungsi, serta pengenalan input/output. Dengan projek ini, diharapkan dapat memahami logika pemrograman dasar, membangun pola berpikir komputasional, dan menyiapkan fondasi untuk mempelajari konsep Python yang lebih lanjut.
