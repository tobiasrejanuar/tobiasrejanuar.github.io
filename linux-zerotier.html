Konfigurasi Remote Server Dengan Zerotier VPN

Konfigurasi ini membahas managemen remote sebuah server dengan menggunakan ZeroTier. ZeroTier adalah solusi jaringan virtual yang memungkinkan koneksi terenkripsi dan aman antara perangkat di mana saja di dunia, seolah-olah mereka berada dalam satu jaringan lokal. ZeroTier menggabungkan keunggulan VPN dan SD-WAN dengan kemudahan penggunaan yang tinggi, menjadikannya solusi ideal untuk individu maupun bisnis yang memerlukan koneksi jaringan yang fleksibel dan aman.

Langkah Konfigurasi

1

Persiapan Awal

Sebelum memulai konfigurasi, pastikan perangkat berikut sudah dipersiapkan :

  • Server dengan sistem operasi linux
  • Akses root atau user dengan privilege sudo
  • Koneksi internet yang cepat & stabil
  • Pastikan sudah membuat akun zerotier terlebih dahulu
  • Aplikasi tambahan seperti zerotier dan termius dibutuhkan jika menggunakan handphone sebagai client
  • Perangkat tambahan seperti handphone/ laptop untuk pengujian
Gambar 1.1: Linux Debian
Gambar 1.2: ZeroTier

Langkah pertama adalah login ke terminal dari server dengan menggunakan ssh lewat konsole (linux) atau putty (windows) :
