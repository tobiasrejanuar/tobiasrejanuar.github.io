Konfigurasi VPN Server Berbasis Debian

Konfigurasi ini membahas implementasi server VPN berbasis Debian 12 menggunakan PPTPD. Server ini dilengkapi aturan firewall, manajemen akses VPN, serta sistem deteksi intrusi untuk memperkuat keamanan jaringan. Dengan konfigurasi tersebut, VPN dapat dimanfaatkan sebagai sarana komunikasi jaringan privat yang aman, menjaga kerahasiaan, integritas, dan ketersediaan data di dalam sistem.

Langkah Konfigurasi

1

Persiapan Awal

Sebelum memulai konfigurasi, pastikan perangkat berikut sudah dipersiapkan :

  • Server dengan sistem operasi linux debian
  • Akses root atau user dengan privilege sudo
Gambar 1.1: Linux Debian
Gambar 1.2: PPTP VPN
2

Login Server Debian

Langkah pertama adalah masuk ke server debian dengan akses root atau user privilege sudo :

Gambar 2.1: Login Server Debian
3

Update & Upgrade Sistem

Langkah selanjutnya adalah update & upgrade sistem agar repository menjadi paling update :

Gambar 3.1: Update Sistem

Commands :

apt update && apt upgrade -y
4

Install Paket PPTPD

Langkah selanjutnya adalah install paket pptpd :

Gambar 4.1: Install Paket PPTPD

Commands :

apt install pptpd -y
5

Buka File pptpd.conf

Langkah selanjutnya adalah membuka file konfigurasi pptpd.conf :

Gambar 5.1: Membuka File pptpd.conf

Commands :

nano /etc/pptpd.conf
6

Konfigurasi File pptpd.conf

Langkah selanjutnya adalah konfigurasi file pptpd.conf, dimana pada bagian baris paling bawah ditambahkan localip dan remoteip :

Gambar 6.1: Konfigurasi pptpd.conf

Commands :

  • Tambahkan di baris paling bawah :
    • localip 193.193.34.1 remoteip 193.193.34.2-5
    7

    Buka File User PPTPD

    Langkah selanjutnya adalah membuka file user pptpd :

    Gambar 7.1: Membuka File User PPTPD

    Commands :

    nano /etc/ppp/chap-secrets
    8

    Konfigurasi User PPTPD

    Langkah selanjutnya adalah konfigurasi user pptpd, dengan menambahkan user & password :

    Gambar 8.1: Konfigurasi User PPTPD
    9

    Buka File DNS User

    Langkah selanjutnya adalah membuka file dns untuk user pptpd :

    Gambar 9.1: Membuka File DNS User

    Commands :

    nano /etc/ppp/pptpd-options
    10

    Konfigurasi File DNS User

    Langkah selanjutnya adalah konfigurasi file dns user pptpd, dengan menambahkan dns di bagian baris bawah # Network and Routing :

    Gambar 10.1: Konfigurasi File DNS User

    Commands :

  • Tambahkan di baris bawah # Network and Routing :
    • ms-dns 193.193.34.1 ms-dns 8.8.8.8
    11

    Restart Service PPTPD

    Langkah selanjutnya adalah restart service pptpd :

    Gambar 11.1: Restart Service PPTPD

    Commands :

    /etc/init.d/pptpd restart
    12

    Mengecek IP Server

    Langkah selanjutnya adalah mengecek dan mencatat ip address server debian yang nanti digunakan untuk tahap pengujian :

    Gambar 12.1: Cek IP

    Commands :

    ip a
    13

    Pengujian

    Langkah terakhir adalah menguji vpn server menggunakan client (windows) yang terhubung satu segmen dengan ip address server :

    Gambar 13.1: Buat Koneksi VPN
    Gambar 13.2: Pengujian

    Tips Penting

    Pastikan sistem sudah ter-update dengan menjalankan sudo apt update && sudo apt upgrade sebelum memulai konfigurasi.
