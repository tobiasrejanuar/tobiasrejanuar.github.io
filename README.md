# Tobias Rejanuar

This repository houses the source code for my personal portfolio and technical documentation site. Built with a focus on efficiency and clarity, this platform serves as a centralized hub for my projects, network configurations, and system administration guides.

## Tech Stack
- **Deployment:** GitHub.
- **Engine:** -
- **DNS & Security:** Cloudflare.
- **Core Topics:** Networking & Linux Systems.

## Project Focus
The website is structured to showcase expertise in:
* **Network Infrastructure:** Configuration of MikroTik, VLAN management, OSPF, PPPoE.
* **Operating Systems:** Advanced administration of Linux distributions, specifically Debian.
* **Service Deployment:** Implementing secure remote access (Cloudflare Tunnels), and containerized environments (Docker).

## Repository Structure
* `/picture` - Assets and architecture diagrams.
* `/projects` - Detailed walkthroughs of implemented technical solutions.
* `index.html` - The main gateway to the portfolio.
* `SECURITY.md` - Security policy and vulnerability reporting procedures.
