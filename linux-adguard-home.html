Konfigurasi Instalasi AdGuard Home Pada Linux

Konfigurasi ini membahas instalasi AdGuard Home pada linux. AdGuard Home adalah perangkat lunak sumber terbuka gratis yang berfungsi sebagai server DNS untuk memblokir iklan dan pelacak di seluruh jaringan. Berbeda dengan ekstensi browser biasa, AdGuard Home melindungi seluruh perangkat yang terhubung ke jaringan Wi-Fi Anda—termasuk Smart TV, konsol game, hingga perangkat IoT tanpa perlu menginstal aplikasi tambahan di setiap perangkat tersebut.

Langkah Konfigurasi

1

Persiapan Awal

Sebelum memulai konfigurasi, pastikan alat dan bahan berikut sudah dipersiapkan :

  • Server dengan sistem operasi linux
  • Akses root atau user dengan privilege sudo
  • Koneksi internet yang cepat & stabil
Gambar 1.1: Linux
Gambar 1.2: AdGuard
2

Login Ke Server

Langkah pertama adalah login ke terminal dari server dengan menggunakan ssh lewat konsole (linux) atau putty (windows) :

Gambar 2.1: Login Ke Server
3

Update & Upgrade Sistem

Langkah selanjutnya adalah update & upgrade sistem agar repository menjadi paling update :

Gambar 3.1: Update Sistem

Commands :

apt update && apt upgrade -y
4

Instal AdGuard Home

Langkah selanjutnya adalah menginstal adguard home pada server :

Gambar 4.1: Menginstal AdGuard Pada Server

Commands :

sudo wget https://static.adguard.com/adguardhome/release/AdGuardHome_linux_amd64.tar.gz
5

Ekstrak File

Langkah selanjutnya adalah mengekstrak file yang telah dinstall sebelumnya :

Gambar 5.1: Ekstrak File

Commands :

sudo tar -xvzf AdGuardHome_linux_amd64.tar.gz
6

Masuk Ke Folder AdGuardHome

Langkah selanjutnya adalah masuk folder AdGuardHome :

Gambar 6.1: Masuk Ke Folder AdGuardHome

Commands :

cd AdGuardHome
7

Jalankan Installer AdGuard Home

Langkah selanjutnya adalah menjalankan installer adguard home. setelah selesai menjalankan installer, copy link yang muncul pada bagian akhir :

Gambar 7.1: Jalankan Installer AdGuard Home

Commands :

sudo ./AdGuardHome -s install

Setting Rule Blocking Adguard Home

Langkah selanjutnya adalah masuk ke halaman dashboard adguard home di website dengan mengisi username & password yang telah dibuat sebelumnya. setelah berhasil masuk, ikuti petunjuk dibawah ini untuk bagian penambahan rule blocking di adguard home :

Gambar 9.1: Masukan Username & Password Yang Telah Dibuat Sebelumnya

10

Setting DNS Pada Router / Modem

Langkah selanjutnya adalah masuk ke menu settingan router / modem (pastikan server adguard home terhubung ke subnet yang sama dengan router), tambahkan ip address dari server adguard home ke dalam menu dns. jika tidak dapat akses settingan router / modem, bisa ditambahkan dns manual pada setiap device yang terhubung :
