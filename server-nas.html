Konfigurasi NAS Server pada Debian Menggunakan Samba dan Web File Manager

Konfigurasi ini membahas implementasi server penyimpanan terpusat (NAS / Network Attached Storage) berbasis Debian 12 yang dapat diakses melalui jaringan lokal. Server ini menyediakan layanan berbagi file menggunakan Samba serta akses berbasis web untuk unggah dan unduh file. Dengan konfigurasi tersebut, NAS dapat dimanfaatkan sebagai media penyimpanan tambahan yang praktis, mudah diakses berbagai perangkat, dan fleksibel untuk dikembangkan lebih lanjut.

Langkah Konfigurasi

1

Persiapan Awal

Sebelum memulai konfigurasi, pastikan perangkat berikut sudah dipersiapkan :

  • Server dengan sistem operasi linux debian
  • Akses root atau user dengan privilege sudo
  • Storage yang cukup
Gambar 1.1: Linux Debian
Gambar 1.2: Samba Server
2

Setting Network

Langkah pertama adalah mengkonfigurasi jaringan, dimana ini menggunakan ip satu segmen dengan router (192.168.1.x) setelah selesai setting simpan dengan ctrl o + ctrl x dan selanjutnya restart :

Gambar 2.1: Setting Network

Commands :

nano /etc/network/interfaces
systemctl restart networking
3

Update & Upgrade Sistem

Langkah selanjutnya adalah update & upgrade sistem agar repository menjadi paling update :

Gambar 3.1: Update Sistem

Commands :

apt update && apt upgrade -y
4

Install Paket Dasar

Langkah selanjutnya adalah install paket dasar yang diperlukan :

Gambar 4.1: Install Paket Dasar

Commands :

apt install sudo vim htop curl wget net-tools unzip ufw fail2ban gnupg2 -y
5

Install Samba Server

Langkah selanjutnya adalah install samba server :

Gambar 5.1: Install Samba Server

Commands :

apt install samba -y
6

Buat Directory

Langkah selanjutnya adalah membuat directory untuk share :

Gambar 6.1: Membuat Directory

Commands :

mkdir -p /srv/nas/public
chmod -R 777 /srv/nas/public
7

Konfigurasi Samba Server

Langkah selanjutnya adalah mengkonfigurasi samba server dan restart jika sudah selesai konfigurasi :

Gambar 7.1: Mengkonfigurasi Samba Server

Commands :

nano /etc/samba/smb.conf
  • Tambahkan di baris paling bawah :
    • [NAS_Public] path = /srv/nas/public browseable = yes writable = yes guest ok = yes create mask = 0666 directory mask = 0777
    systemctl restart smbd
    8

    Install Nginx

    Langkah selanjutnya adalah Install nginx dan aktifkan setelahnya :

    Gambar 8.1: Install Nginx

    Commands :

    apt install nginx php-fpm php-zip php-mbstring php-curl -y
    systemctl enable nginx --now
    9

    Install File Browser

    Langkah selanjutnya adalah Install File Browser :

    Gambar 9.1: Install File Browser

    Commands :

    curl -fsSL https://raw.githubusercontent.com/filebrowser/get/master/get.sh | bash
    10

    Membuat Directory Data

    Langkah selanjutnya adalah Membuat Directory Data :

    Gambar 10.1: Membuat Directory Data

    Commands :

    mkdir -p /srv/nas/web
    11

    Menjalankan File Browser

    Langkah selanjutnya adalah menjalankan file browser :

    Gambar 11.1: Menjalankan File Browser

    Commands :

    filebrowser -r /srv/nas/public --address 0.0.0.0 --port 8080 &
    12

    Pengujian

    Langkah terakhir adalah menguji dengan akses lewat browser http://192.168.1.25:8080 dengan username = admin dan password yang tertera saat menjalankan file browser :

    Gambar 12.1: Pengujian Login
    Gambar 12.2: Pengujian File

    Tips Penting

    Pastikan sistem sudah ter-update dengan menjalankan sudo apt update && sudo apt upgrade sebelum memulai konfigurasi.
